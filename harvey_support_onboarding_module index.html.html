3Common Ticket Scenarios

Know It Before It Hits the Queue

These are the most common support patterns across Harvey's customer base. Click each to see the recommended approach. Your enablement content should make these instinctive.

🧑‍🏫
Sam, QA Lead
About 40% of escalations come from one of these five patterns. If a specialist knows these cold, escalation rates drop — fast. Your job is to make sure the content makes that happen.
"Harvey's answer doesn't cite a source — is it hallucinating?" ▼
This is the #1 anxiety for new users in legal. Clarify: Harvey cites sources when grounding its answer in documents or legal databases. For general reasoning or summarization tasks, responses are generated — not retrieved. Guide users to the Knowledge product for citation-heavy research workflows.
"I uploaded a document to Vault but can't find it." ▼
Usually a permissions or tagging issue. Check whether the user is in a Shared Space vs. a personal workspace. Documents uploaded to a shared space may not appear in individual search without the right access. Walk them through the Vault filter panel — most users haven't explored it yet.
"My Harvey Agent stopped mid-task — what happened?" ▼
Agents can pause on ambiguous inputs or missing context. Check the Agent's task log in Command Center — it will show exactly where it stopped and why. This is also a great trigger to create a quick-reference guide on how to write clearer agent prompts for their firm's workflows.
"How do I know if our team is actually using Harvey?" ▼
Point them to Command Center — it's built exactly for this. Adoption metrics, usage by feature, and benchmarking against peer firms are all available there. If they're not seeing data, check that their admin has enabled firm-level reporting.
"Can Harvey review contracts in languages other than English?" ▼
Yes — Harvey supports multilingual contract review through Contract Intelligence. Performance varies by language and document complexity, so set accurate expectations. Refer them to the Harvey Academy for a walkthrough on optimizing multilingual workflows.
